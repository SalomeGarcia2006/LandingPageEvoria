# EVORIA - Sistema de Logística y Planificación de Eventos

![EVORIA](logoevoria-removebg-preview.png)

## 📖 Descripción

EVORIA es un sistema web para la planificación y ejecución de eventos, gestionando recursos humanos, inventario, proveedores, presupuestos y cronogramas en una sola plataforma.

---

## 🚀 Características

| Módulo | Descripción |
|--------|-------------|
| 📅 Gestión de Eventos | Creación y seguimiento de eventos |
| 👥 Recursos Humanos | Asignación y control de personal |
| 📦 Inventario | Control de recursos materiales |
| 💰 Presupuestos | Cálculo automático de costos |
| ⏰ Cronogramas | Generación de actividades y horarios |
| 🔔 Alertas y Reportes | Notificaciones y exportación PDF |

**Landing Page:**
- Gradiente azul difuminado (oscuro → claro)
- Partículas animadas interactivas (respuesta al mouse)
- Diseño responsive (mobile/tablet/desktop)
- 8 Casos de Uso documentados

---

## 🛠️ Tecnologías
